# Корпус знаний

В этом каталоге хранится корпус знаний проекта `openclaw-skills`.

Корпус инициализирован и содержит зарегистрированные источники по OpenClaw
skills и связанным внешним материалам.

Базовые файлы:

- `corpus.yml` — договор структуры и правил хранения;
- `catalog.yml` — реестр источников;
- `index/` — сводные индексы по единицам источников и утверждениям;
- `data/` — карточки источников и их артефакты;
- `.local/` вне Git — сырое, приватное и временное содержимое.

Текущий слой обработки содержит нормализованные материалы и извлечённые
проверяемые утверждения для зарегистрированных источников.
