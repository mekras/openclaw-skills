# openclaw-skills

`openclaw-skills` — APM-проект для подключения набора агентских навыков и
правил `ai-dev-team` в локальное окружение Codex/OpenClaw.

Проект хранит минимальную конфигурацию APM: зависимость от `ai-dev-team`, файл
блокировки и сгенерированные инструкции для агентов. Основной пользователь
README — сопровождающий репозитория, которому нужно воспроизвести установку и
обновить инструкции после изменения зависимостей.

## Быстрый старт

Установите зависимости APM из файла блокировки:

```bash
apm install --frozen
```

После установки пересоберите инструкции для активной цели Codex:

```bash
apm compile --target codex
```

Команда обновляет `AGENTS.md` и доставляемые через APM файлы в `.agents/` и
`.codex/`. Эти каталоги остаются локальными артефактами установки и не
отслеживаются в Git.

## Сопровождение

Проверить доступные цели APM:

```bash
apm targets
```

Обновить зависимости в пределах ограничений `apm.yml`:

```bash
apm update
```

После обновления проверьте изменения в `apm.lock.yaml` и заново выполните
компиляцию инструкций.
